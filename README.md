# Selenium Automation Testing Framework

A Hybrid Automation Framework developed using Java, Selenium WebDriver, TestNG, Cucumber (BDD), Maven, and Page Object Model (POM).

Supports data-driven testing and detailed reporting using Extent Reports.

---

##  Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Cucumber
- Maven
- Extent Reports
- Git & GitHub
- JSON

---

##  Key Features

 End-to-end e-commerce automation  
 Cucumber BDD implementation  
 Page Object Model  
 Data-driven testing  
 Extent Reports integration  
 Retry & Listener support  

---

##  How to Run

1. Clone the repository
2.  Run tests
3.  
OR

Run `TestNGTestRunner.java` from IDE.

---

##  Reports

Execution reports are generated using Extent Reports in the `reports` folder.
