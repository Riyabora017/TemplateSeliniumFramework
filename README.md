# Selenium Automation Testing Framework

Designed and developed a scalable hybrid automation framework using Java, Selenium, TestNG, Cucumber, and Maven to automate functional and regression testing of web applications.

Implemented industry best practices to ensure maintainability, reusability, and reliable reporting.

---

## Project Highlights

Built a modular and reusable automation framework from scratch

Designed Page Object Model (POM) for better code structure

Implemented data-driven testing using JSON files

Integrated Extent Reports for detailed execution analysis

Automated critical e-commerce workflows (login, cart, checkout, order)

Used Git & GitHub for version control and team collaboration

##  Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Cucumber
- Maven
- Extent Reports
- Git & GitHub
- JSON

---

##  Key Features

Hybrid framework (TestNG + Cucumber + POM)

Parallel execution support

Retry mechanism for failed test cases

Custom TestNG Listeners

Config-driven execution (config files)

Cross-browser execution support (if applicable)

Easy scalability for new test cases
